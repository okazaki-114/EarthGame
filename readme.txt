■ ゲーム説明
同じ惑星を重ね合わせ次の惑星へ変化させます。
変化させた際スコアが加算されます。
ハイスコアを目指してプレイをお楽しみください。

[進化の流れ]
冥王星 → 月 → 水星 → 火星 → 金星 → 地球 → 海王星 → 天王星 → 土星 → 木星 → 太陽

[点数表]
3の累乗で点数が加算されています。

点数表：
　　　　冥王星 → 月　　 ＝ 1
　　　　月　　 → 水星　 ＝ 3
　　　　水星　 → 火星　 ＝ 9
　　　　火星　 → 金星　 ＝ 27
　　　　金星　 → 地球　 ＝ 81
　　　　地球　 → 海王星 ＝ 243
　　　　海王星 → 天王星 ＝ 729
　　　　天王星 → 土星　 ＝ 2187
　　　　土星　 → 木星　 ＝ 6561
　　　　木星　 → 太陽　 ＝ 19683
　　　　太陽　 → 消滅　 ＝ 59049

[ゲームオーバー条件]
ステージの上にあるラインに惑星が5秒間接している場合ゲームオーバーとなります。
